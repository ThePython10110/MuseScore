# MuseScores
My MuseScore scores, also available [here](https://www.musescore.com/thepython10110).

Since MuseScore doesn't allow you to download files without a Pro account, I've decided to make mine publicly available on GitHub. I also wrote a Python script that will automatically export all of them to various filetypes (and only if the MuseScore files have been modified).

If you're going to distribute these, just make sure to give me (ThePython or ThePython10110) proper credit.
